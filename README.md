
# Simple Task Manager

A simple Android app to manage daily tasks.  
Built with Kotlin and Gradle (KTS).

## Features

- View all tasks
- Add a new task
- Edit an existing task
- Mark tasks as complete with checkboxes

3. Build and run on emulator or device
